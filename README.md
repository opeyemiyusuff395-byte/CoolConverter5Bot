# CoolConverter5Bot

A multipurpose Telegram bot that can:
- Convert images to different formats (JPG, PNG, WebP, BMP, TIFF, GIF)
- Shorten URLs using spoo.me API
- Count words, characters, sentences, and paragraphs

## Deployment
This bot is ready to deploy on Railway.app via GitHub.

## Environment Variables
- `TELEGRAM_BOT_TOKEN`: Your bot token from @BotFather

## Commands
- `/start` - Welcome message
- `/help` - Help message
- Just send an image, URL, or text to use the bot!
